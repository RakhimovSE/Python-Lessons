ФОРМАТИРОВАНИЕ

1. Форматирование с подстановкой по номеру параметра
Пример: "{0} + {1} = {2}".format(1, 2, 3)
Вывод: "1 + 2 = 3"

2. Форматирование с подстановкой по названию параметра
Пример: "{a} + {b} = {c}".format(a=1, b=2, c=3)
Вывод: "1 + 2 = 3"

3. Форматирование с указанием типа переменной
%d - целочисленная переменная
%f - дробное число. Также можно указать точность вывода (кол-во знаков после запятой)
%.2f - вывод дробного числа с 2 знаками после запятой
%s - строковая переменная

Пример: "%d + %d = %d" % (1.57, 2, 3)
Вывод: "1 + 2 = 3"

Также можно задавать кол-во символов, отведенных для вывода числа. Это полезно при выводе матриц
Пример: "%4d: %6s" % (2, "Hi")
Вывод: "___2: ____Hi" (пробелы, выведенные через форматирование, обозначены как "_")
